# AI Lead Scoring & CRM Alert — n8n Automation

Automatically captures incoming leads, scores them, saves to Airtable, and fires a Gmail alert instantly when a hot lead comes in.

## What it does
- Receives leads from any web form via webhook
- Scores each lead based on company size and message intent
- Saves every lead to Airtable CRM automatically
- Fires an instant Gmail alert if score is 7 or above
- Cold leads saved silently — no noise, no missed opportunities

## Built with
- n8n
- Airtable API
- Gmail OAuth2
- JavaScript

## The problem it solves
Most small businesses lose leads because they respond too slowly. This runs 24/7 and alerts you within seconds of a hot lead coming in.

## Setup time for a client
1 day. Connects to any existing form or website.
